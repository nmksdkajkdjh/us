## 🚨 已知限制

### 当前限制

```
1. ZIP数据库大小
   - 包含主要城市 (可按需扩展至41k+)
   - 如需完整的41k+ ZIP码，参考指南更新

2. Google Places免费配额
   - 每月2500个会话免费
   - 超额后自动回退到本地建议

3. 地址格式
   - 仅支持美国地址
   - 可扩展国际地址 (待未来版本)

4. 离线限制
   - GPS需要网络定位
   - Google Places需要网络（本地建议可离线）
```

### 解决方案

```
1. ZIP数据库扩展 → ADVANCED_FEATURES_GUIDE.md
2. Google配额管理 → 设置速率限制或升级计划
3. 国际支持 → 预留在v3.0规划中
4. 离线模式 → 使用本地建议作为备选
```
